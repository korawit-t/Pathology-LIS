# Terms of Use & Support Policy

## 1. บริบทของซอฟต์แวร์ / Software Nature

ซอฟต์แวร์นี้เป็นระบบสารสนเทศทางห้องปฏิบัติการพยาธิวิทยากายวิภาค (Pathology LIS) เผยแพร่ในรูปแบบ Open Source และส่งมอบในลักษณะ "As-Is" เพื่อให้นำไปติดตั้ง บริหารจัดการ และใช้งานภายในองค์กรของตนเอง ซอฟต์แวร์นี้ไม่ถือเป็น Medical Device และไม่ได้ผ่านการรับรองจากหน่วยงานกำกับดูแลทางการแพทย์ใดๆ

> This software is an open-source Pathology Laboratory Information System (LIS), provided "as-is" for hospitals to install, manage, and operate within their own organization. It is not a Medical Device and has not been certified by any medical regulatory authority.

## 2. ขอบเขตความรับผิดชอบของฝ่ายไอทีโรงพยาบาล / Hospital IT Responsibilities

เพื่อให้ระบบทำงานได้อย่างต่อเนื่อง ฝ่ายไอทีของโรงพยาบาลต้องรับผิดชอบในส่วนต่อไปนี้:

- **Infrastructure** — การจัดเตรียม Server, ระบบ Network และความปลอดภัยของ Firewall
- **Data Backup** — การสำรองข้อมูลฐานข้อมูลเป็นประจำ เพื่อป้องกันข้อมูลสูญหาย
- **Client Support** — การติดตั้งโปรแกรมและแก้ไขปัญหาการใช้งานเบื้องต้นให้แก่ผู้ใช้งาน
- **System Integration** — การดูแล Interface และการตรวจสอบสถานะการรับ-ส่งข้อมูลระหว่าง LIS และ HIS ของโรงพยาบาล

> To keep the system running reliably, the hospital's IT department is responsible for: server infrastructure, network, and firewall security; regular database backups; end-user support and client setup; and monitoring the LIS–HIS integration.
